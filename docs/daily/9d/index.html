 尽管每天都感觉很忙，但回头一想，却觉得啥也没干，每天就在这样的焦虑中度过。

 为了解决这个问题，我决定学着做一下时间管理，就像我们工作那样，给自己写OKR、分Sprint、设置番茄钟。今天是第一天开始执行，为了响应这样的决定我还做了一些仪式感——在聊天群里设置自己的昵称为「升级中...」，暗示自己在新的六月要摆脱五月的颓势，重新起步。

 然而万事开头难，中间难，最后更难。眼看着一个个番茄钟过去，我却不知道干什么，看看IM信息，协助一下同事工作，做一些根本进不了记忆的琐事，这一天就那么到头了！

 我意识到时间管理之后，我还欠缺目标管理。说到底如果我没找到该做的事情的话，不打游戏挤出来的时间也会变成聊天看剧。所以接下来，必须得把该做的事情排一排才行。

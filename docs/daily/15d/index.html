 因为最近瞎买了些数字货币，亏得一塌糊涂，想起一件事情：巴菲特99.8%的资产是在他50岁后赚到的。

 顿时发觉自己心态不对，竟鬼迷心窍想着赚快钱！也应征了巴哥说的那句「因为没有人愿意慢慢变富」。

 喝了一口马蹄水之后我老觉得哪里不对劲。50岁？巴哥50岁之后才发达，赚那么多钱他都用来干啥呢？！我现在之所以那么想暴富，是因为我欲望还在，我想要最新款的MacBook；想要一万九的单反；因为长期面对显示器，颈椎疼得要命，我想要一把上万块的hermanmiller人体工学椅；618马上就要到了，我想要买最新款IPhone！在我的购物车里还有机械键盘、音箱、衣服鞋子...但是如果我50岁了，我买什么呢？

 无趣啊！所谓七宗罪之嫉妒，大概就是一个人过于往别人身上寻找自己所缺失，而忽视了自己于别之盈饶，郁闷来得静悄悄。

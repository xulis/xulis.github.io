 偶然听到以前常听的一首歌(Politic Amagni " La politique, c'est pas bon")，音符响起时记忆也翻滚而来，是落网！我甚至记得那一期音乐推荐的标题叫「长路迢迢」，甚至其配图也是记忆犹新：一个手握长矛的非洲部落长者，面对着夕阳西下，似是已经历了长路迢迢，又似是要倚着那瘦弱的身躯开始长路迢迢。

 如今的音乐App恐怕不会为我推送这样的内容了。应用后面的算法是没有感情的，它只能给我贴标签，然后按部就班地筛选那些有同样标签的音乐而已。

 以Up主推荐形式的音乐网站，还有伴随我青春的Songtaste，他们都一去不复返了。

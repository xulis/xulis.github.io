 今天 Sam 说他要离职了。我已经不记得上一次见他是什么时候了，但是他今天找我聊天的时候让我想起很多事情。

 跟别的领导不一样，Sam 与下属的交流很多时候是仰仗他庞大的知识库，至少我们的交流是这样的。他会给我们讲外星人、量子力学、柏拉图、行为认知……他喜欢刷得到的课程。总之，他似乎像是什么都懂，像是百科全书，精力也让人感觉永远是饱满的，对外界充满了求知欲，我喜欢跟他聊天，因为跟他聊天，我可以对任何事情提出「为什么」可以产生思考的交流是非常有价值的，对于远程办公的我来说，与人产生有价值的交流的机会并不多。

 最后，祝好吧。
